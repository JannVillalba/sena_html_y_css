Glosario Tecnológico Parte 2

← A-N O P Q R S T U V W X Y Z

O

Open Source: Software cuyo código fuente está disponible públicamente para su modificación y distribución.

OS (Sistema Operativo): Software que gestiona los recursos hardware y provee servicios a los programas.

P

Python: Lenguaje de programación interpretado de alto nivel y propósito general.

Protocolo: Conjunto de reglas que definen cómo se comunican los dispositivos en una red.

Q

Query: Consulta para recuperar o manipular datos en una base de datos.

Quantum Computing: Computación que utiliza qubits para realizar operaciones.

R

React: Biblioteca JavaScript para construir interfaces de usuario.

Responsive Design: Diseño web que se adapta a diferentes tamaños de pantalla.

S

SQL: Lenguaje de consulta estructurado para bases de datos relacionales.

SSL (Secure Sockets Layer): Protocolo de seguridad para comunicaciones en internet.

T

TypeScript: Superset tipado de JavaScript que compila a JS estándar.

TCP/IP: Conjunto de protocolos para la comunicación en redes.

U

UI (Interfaz de Usuario): Medio por el cual un usuario interactúa con un dispositivo.

UX (Experiencia de Usuario): Experiencia global del usuario al interactuar con un producto.

V

Virtualización: Creación de versiones virtuales de recursos informáticos.

VPN (Red Privada Virtual): Conexión segura a través de internet.

W

WebAssembly: Formato de código binario para ejecución en navegadores.

Wi-Fi: Tecnología para redes inalámbricas locales.

X

XML (Extensible Markup Language): Lenguaje de marcado para almacenar datos.

XSS (Cross-Site Scripting): Vulnerabilidad de seguridad web.

Y

YAML: Lenguaje de serialización de datos legible por humanos.

Z

Zero-Day: Vulnerabilidad desconocida para el desarrollador.

ZIP: Formato de compresión de archivos.
