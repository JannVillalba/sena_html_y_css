A

Algoritmo: Conjunto de instrucciones o reglas definidas para resolver un problema o realizar una tarea específica.

API (Interfaz de Programación de Aplicaciones): Conjunto de definiciones y protocolos que permiten la comunicación entre diferentes aplicaciones.

B

Backend: Parte de una aplicación que se encarga de la lógica del servidor y la gestión de bases de datos.

Big Data: Conjunto de tecnologías y técnicas para el procesamiento y análisis de grandes volúmenes de datos.

C

Cloud Computing: Modelo de computación que permite el acceso a recursos informáticos a través de Internet.

CSS (Cascading Style Sheets): Lenguaje utilizado para describir la presentación de un documento HTML.

D

DevOps: Conjunto de prácticas que combinan el desarrollo de software y las operaciones de TI para mejorar la colaboración y la eficiencia.

Docker: Plataforma que permite crear, desplegar y ejecutar aplicaciones en contenedores.

E

Estructura de Datos: Forma de organizar y almacenar datos para facilitar su acceso y manipulación.

Extensión de Archivo: Sufijo al final de un nombre de archivo que indica su formato o tipo (ej. .txt, .jpg).

F

Frontend: Parte de una aplicación con la que interactúa el usuario, desarrollada con tecnologías como HTML, CSS y JavaScript.

Framework: Conjunto de herramientas y librerías que facilitan el desarrollo de software al proporcionar una estructura base.

G

Git: Sistema de control de versiones distribuido para rastrear cambios en el código fuente.

GPU (Unidad de Procesamiento Gráfico): Componente especializado en el procesamiento de gráficos y operaciones paralelas.

H

HTML (HyperText Markup Language): Lenguaje estándar para crear y estructurar páginas web.

HTTP (Hypertext Transfer Protocol): Protocolo de comunicación utilizado para transferir datos en la web.

I

IDE (Entorno de Desarrollo Integrado): Software que combina herramientas de desarrollo en una sola interfaz (ej. Visual Studio, Eclipse).

IoT (Internet of Things): Red de dispositivos físicos conectados a Internet que recopilan e intercambian datos.

J

JavaScript: Lenguaje de programación utilizado principalmente para desarrollo web frontend y backend (Node.js).

JSON (JavaScript Object Notation): Formato ligero de intercambio de datos basado en texto, fácil de leer y escribir.

K

Kernel: Núcleo de un sistema operativo que gestiona los recursos del hardware y permite la comunicación entre software y hardware.

Kubernetes: Plataforma de orquestación de contenedores para automatizar la implementación y escalabilidad de aplicaciones.

L

Lenguaje de Programación: Conjunto de reglas y sintaxis para escribir instrucciones que una computadora puede ejecutar.

Linux: Sistema operativo de código abierto basado en Unix, ampliamente utilizado en servidores y desarrollo.

M

Machine Learning: Campo de la inteligencia artificial que permite a los sistemas aprender y mejorar automáticamente con experiencia.

Metadatos: Datos que describen otros datos, proporcionando información sobre su estructura, origen o contexto.

N

Node.js: Entorno de ejecución de JavaScript en el servidor, basado en el motor V8 de Chrome.

NoSQL: Tipo de base de datos no relacional que almacena datos en formatos flexibles (documentos, grafos, etc.).
